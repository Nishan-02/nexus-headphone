# Nexus Headphone Backend

Simple Express.js backend for the Nexus Headphone project.

## Features
- GET `/api/products`: Fetch all products
- GET `/api/products/:id`: Fetch a single product
- POST `/api/orders`: Place a new order

## Setup
1. `cd backend`
2. `npm install`
3. `npm run dev` (starts on http://localhost:5000)

## Tech Stack
- Express.js
- CORS
- dotenv
- nodemon (development)
